# Tensor Networks for Efficient Representation and Compression of Large Language Models

**Author:** Ruwail Hassan

A literature review examining tensor networks as a mathematically principled 
approach to LLM compression, tracing their origins in quantum many-body physics 
to modern applications in transformer architectures.

This was written as a coursework assignment for NSCI0004 at University College London 
with a 3,000-word limit, so coverage of certain topics is necessarily brief.
